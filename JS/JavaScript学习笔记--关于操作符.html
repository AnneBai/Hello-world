truthy和falsy;
truthy：转换为布尔值时为true的；
falsy：转换为布尔值时为false的；

布尔操作符：
逻辑非一定返回布尔值；
逻辑与 和 逻辑或不一定返回布尔值，且均为短路操作符；
逻辑与：判断第一个操作数，falsy的情况包括6种：数值0，布尔值false，null, undefined, NaN, 空字符串"";
只有当第一个操作数为truthy的情况下，返回第二个操作数；
逻辑或：只要第一个操作数为truthy，就返回第一个操作数；第一个操作数为falsy的情况下，返回第二个操作数；

避免取未定义或不存在的值报错：
《Javascript 语言精粹》P21:

尝试从undefined的成员属性中取值将会导致TypeeError异常，这时可以通过“&&”操作符来避免错误:
flight.equipment // undefined
flight.equipment.model //throw"TypeError"
flight.equipment && flight.equipment.model //undefined

递增和递减操作符语句如果外部还有其他操作符，若为前置递增和递减操作，则整句被从左到右依次执行；
若为后置递增和递减操作，则是在其所在的外部语句求值之后再执行；

var num1 = 2; var num2 = 20; var num3 = num1++ +num2; //22 var num4 = num1+num2; //23 如果写在num3，则为22

递增或递减对操作数进行再赋值，会覆盖原来的值；

字符型数值前用“+”（或“-”变为负值）可以转变为数值类型，但只在本语句中有效；

字符型数值直接参与+或-运算则仍为字符型；

var num1 = '2'; var num2 = 20; var num3 = ++num1+num2; //23 var num4 = num1+num2;//23 var num5 = num2+num1;//23 num1 //3 typeof num1 //number var num1 = '2'; var num2 = 20; var num3 = +num1+num2; //22 var num4 = num1+num2;//220 var num5 = num2+num1;//202 num1 //'2' typeof num1 //string
