关于函数中的 return
函数在定义时不必指定是否返回值，任何函数在任何时候都可以通过return语句后跟要返回的值来实现返回值；
执行完return语句会立即退出，并返回对应的结果，其之后的语句不会执行（同一个函数内）。
return语句也可以不带任何返回值，一般用在需要提前停止函数执行而又不需要返回值的情况下。
return false;
  1 取消默认行为；最常见的是点击一个链接会跳转页面；如果在链接绑定的onclick事件处理函数中设置return false，则不会跳转页面。
  2 作为一个会返回false值的函数，如果是内嵌在一个链接/图片等的事件处理函数中的，那么也会取消其链接对应的默认跳转页面的行为。
return true;
  返回true，对于链接来说不会取消默认行为；
return this;
  返回调用函数的对象本身，这个可以用于链式编程，在调用函数之后可以继续访问方法或属性而不用另起一行；
